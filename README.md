## Installation

```bash
# Cloné le dépot
git clone https://github.com/pierretl/x1-alpha.git

# On se place dans le répertoire
cd x1-alpha/

# Installer les dépendances du projet
npm install -y
```

## Build & Run

|Commande|Effet|
|--------|-----|
|`gulp build`|Build|
|`gulp watch`|Run + lancement serveur local|
